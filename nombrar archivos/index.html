Pautas para nombrado de archivos

  1. NO usar espacios. Aunque el sistema operativo lo permita, esto dificulta el tratamiento automático.
    2. 
         ❌ guia de anotacion.txt

         ✅ guia-de-anotacion.txt
            guia_de_anotacion.txt
            guiaDeAnotacion.txt
  2. NO usar caracteres especiales:
    • no usar acentos
    • no usar caracteres que no pertenezcan al alfabeto inglés ('ñ', 'ç', etc.)
    • no usar & ˜ ! @ # $ % ^ * () ` ; < > ? , [] {} ' " / |

    Sí se pueden usar:

    • números
    • puntos
    • mayúsculas

  3. Al numerar una secuencia de archivos, usar la cantidad necesaria de ceros al inicio para que los archivos sean ordenados correctamente.
    4. 
         ❌ cap1.txt cap10.txt cap11.txt cap2.txt cap3.txt etc.

         ✅ cap001.txt cap002.txt cap003.txt ... cap010.txt cap011.txt ... cap100.txt
  4. Al usar fechas, preferir el formato YYYYMMDD o YYMMDD. Así los archivos serán ordenados cronológicamente.

  5. Usar nombres descriptivos que permitan identificar fácilmente el contenido
    6. 
         ❌ guia1.txt

         ✅ guia_anotacion_Pelagios.v1.txt
  6. Usar nombres concisos. Los nombres largos dificultan el tratamiento con ciertos programas.

  7. Ser consistente con el formato elegido.
    8. 
          ❌ ALVAR-Comentarios.txt 
             AlvarNunez-Naufragios.txt 
             dubiscay_archiveorg.txt 
             laargentina2017.TXT 
             ulrico.txt
          
          ✅ AlvarNunez-Comentarios.txt 
             AlvarNunez-Naufragios.txt 
             DuBiscay-Informe-de-un-viaje.txt 
             RuiDiaz-la-Argentina-manuscrita.txt 
             Schmidel-Viaje.txt
